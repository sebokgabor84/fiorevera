# Cinematic Animation Directive: "Liquid Glass" Steampunk

Master directive for converting static QA background assets into seamless, hypnotic living wallpapers.

## Movement Constraints
- **Camera State**: Strictly Static (0 pan, 0 tilt, 0 zoom, 0 shake).
- **Style**: macOS Sequoia "Liquid Glass" aesthetic — organic and rhythmic.

## Motion Instruction Matrix

| Element | Animation Rule | Visual Quality |
|---|---|---|
| **Liquid Gold** | Slow, graceful flow with internal turbulence. | High refraction, viscous fluid physics. |
| **Amber Filaments**| Rhythmic "breathing" pulse in brightness. | Soft atmospheric glow, no frequency flash. |
| **QA Display** | Perfectly static with high-frequency scan-flicker. | Vintage CRT/phosphor phosphor look. |
| **Deep Gears** | Barely perceptible rotation of background brass. | Mechanical depth, working-machine feel. |
| **QR Frame** | **100% Static and sharp**. | Guaranteed future scan-ability. |

## Technical Specifications
- **Output**: 4K, 60fps, Seamless Loop.
- **Quality**: Ultra-high fidelity, zero morphing, zero artifacts.
