# Desktop Background Generator

Generates "Artisan Tahoe" style mechanical steampunk interfaces optimized for high-resolution displays (8K) and specific hardware ratios (16:10).

## When to use this skill
- Generating custom desktop backgrounds for QA stations.
- Creating wallpapers that adhere to the "Midnight Luminescence" project aesthetic.
- Developing assets with specific functional placeholders (e.g., QR codes, terminal displays).

## Visual Identity Tokens

| Element | Specification |
|---|---|
| **Branding** | Vintage green-phosphor digital display ('QA' text, 1980s terminal font). |
| **Functional Space**| Square framing module (upper-right) in polished copper/beveled glass for QR codes. |
| **Materials** | Deep Polished Copper, Brushed Gold, High-gloss Mahogany. |
| **Lighting** | Amber-neon vacuum tube filaments, liquid-gold energy conduits. |
| **Screen Spec** | MacBook Pro 13" (16:10), 8K Optimized resolution. |

> "Extreme macro close-up of an intricate steampunk QA testing station at midnight. In the upper right quadrant, there is a dedicated square framing module made of polished copper and dark beveled glass, designed as a placeholder for a QR code. In the mid-left area, a small integrated glowing green-phosphor digital screen clearly displays the letters 'QA' in a vintage computer font. High visual depth with cinematic bokeh. Surfaces show subtle textures of brushed metal and copper rivets. Cinematic 16:10 widescreen composition, 8K resolution, photorealistic, master craftsman quality."

## Best Practices / Constraints
- **Hardware-Specific**: Always verify the aspect ratio (16:10) for MacBook Pro compatibility.
- **Material Integrity**: Ensure "liquid gold" and "polished copper" maintain the Steampunk aesthetic described in `design-system-expert`.
- **Placeholder Safety**: The QR code module must remain a clean, beveled area to avoid scanning interference when the asset is used as a background.
